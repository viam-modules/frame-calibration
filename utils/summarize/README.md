### Instructions

Run `make create-example-config`.
Populate the config with the necessary information.

### Experimental setup.

Orient your april tag sheet such that april tag 16 is facing up. In this orientation april tag 11 should be the right of april tag 16 and april tag 21 should be to the left of april tag 16.

Orient april tag 16 such that its Y value position is as close as possible to zero.

Refer to apriltagsetup.png to see our experimental setup.
